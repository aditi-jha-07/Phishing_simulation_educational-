# 🎣 Phishing Hunter

A cybersecurity education platform designed to train students in 
identifying phishing emails through interactive simulations.

## 🎯 Purpose

Phishing Hunter is a web-based simulation tool built for academic 
environments. Students practice recognizing phishing emails in a 
safe, controlled setting — while administrators monitor performance 
and manage scenarios in real time.

## 👥 Who It's For

- **Students** (.edu email required) — Complete simulations, review 
  results, and sharpen their phishing detection skills
- **Instructors / Admins** — Monitor all student activity, view 
  analytics, and manage the email scenario library

## ⚙️ Tech Stack

- **Frontend:** React, TailwindCSS
- **Backend:** Node.js, Express.js
- **Database:** MySQL (auto-initialized on first run)
- **Auth:** JWT + bcrypt, role-based access control

## ✨ Key Features

- 🔐 Secure registration and login with role-based access
- 📧 Realistic phishing email simulations (10 scenarios per session)
- ⚡ Real-time answer recording and scoring
- 📊 Admin dashboard with live student performance analytics
- 🛠️ Full scenario management (add, edit, delete)
- 🗄️ Auto-creates all database tables on first run
